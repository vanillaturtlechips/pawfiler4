# AI 영상 분석

PawFiler의 AI 영상 분석 기능은 사용자가 직접 업로드한 영상의 딥페이크 여부를 분석해 줍니다.

## 분석 방식

멀티모달 방식으로 3가지 측면을 종합 분석합니다:

1. **시각 분석**: 영상의 얼굴, 피부, 조작 흔적 등 시각적 요소 분석
2. **음성 분석**: 음성 딥페이크 여부 탐지
3. **립싱크 분석**: 음성과 입 모양의 일치 여부 확인

## 분석 결과

- **신뢰도 점수**: 딥페이크 여부에 대한 확률 점수 제공
- **조작 의심 구간**: 영상에서 조작이 의심되는 특정 구간 정보 제공

## 결과 공유

분석 결과는 커뮤니티(광장)에 공유할 수 있습니다.
