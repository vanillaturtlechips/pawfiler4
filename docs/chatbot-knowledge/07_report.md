# 리포트 서비스

PawFiler의 리포트 서비스는 사용자의 퀴즈 플레이 데이터를 분석해 개인 맞춤형 HTML 리포트를 생성해 줍니다.

## 리포트 다운로드 방법

통계 탭에서 리포트 다운로드 버튼을 클릭하면 리포트를 받을 수 있습니다.

## 리포트 포함 내용

- **전체 정답률**: 전체 퀴즈 정답률 통계
- **퀴즈 유형별 통계**: easy/medium/hard 난이도별 성적 분석
- **자주 틀린 영역**: 반복적으로 틀린 유형이나 패턴 분석
- **주간 정답률 추이**: 주별 성적 변화 그래프

## 주의 사항

리포트 생성에는 다소 시간이 걸릴 수 있습니다.
