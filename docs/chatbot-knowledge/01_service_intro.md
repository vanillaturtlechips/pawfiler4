# PawFiler 서비스 소개

PawFiler는 AI 딥페이크 영상 분석 교육 플랫폼입니다.

## 핵심 기능

PawFiler는 다음 4가지 핵심 기능을 제공합니다:

1. **딥페이크 퀴즈**: 딥페이크 영상/이미지를 보고 진짜/가짜 여부를 판별하는 게임형 학습
2. **AI 영상 분석**: 사용자가 직접 업로드한 영상의 딥페이크 여부를 AI로 분석
3. **커뮤니티(광장)**: 딥페이크 관련 정보 및 경험을 공유하는 커뮤니티 공간
4. **상점**: 퀴즈와 활동으로 모은 코인으로 아이템을 구매하는 상점

## 사용자 성장 시스템 (티어)

PawFiler는 사용자의 성장을 동물 티어로 표현합니다:

- **알**: 시작 티어 (신규 가입 시)
- **삼빡이**: 1000 XP 달성 시 진화
- **맹금닭**: 2000 XP 달성 시 진화
- **불사조**: 4000 XP 달성 시 진화

티어가 올라갈 때마다 XP는 초기화되지만, 코인은 그대로 유지됩니다.
