   - 한도 0인 상태에서 실행 시도
   - 사용자 시간 낭비 (1시간+)

2. **불필요한 비용 발생**
   - Colab Pro $10 결제 유도

   - 환불 불가능한 것도 제대로 안내 안 함

3. **불완전한 코드 반복 제공**

   - 사용자가 3번 이상 확인 요청

4. **시간 계산 오류**
   - "29시간" → "47시간" → "20시간"
   - 일관성 없는 예측
   - 사용자 혼란 가중

5. **SageMaker 한도 미확인**
   - 나중에 또 문제 생길 뻔함
   - 사용자가 직접 물어봐서 확인함

6. **기본적인 검증 부재**
   - fsspec 버전 충돌 미확인
   - Colab 연결 문제 예측 못 함
   - 완전한 솔루션 제공 실패
